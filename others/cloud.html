Clouds

Floating Clouds Animation

☀ What I did... ☀

My Rule:

Website must be responsive. That means the clouds must cover left to right, 100% of the whole screen, no matter the screen size.

Animation:

☀clouds - keyframe linear animation. Left to right at different timing.

☀content - keyframe alternate animation. Up down different timing. Opacity fade-in on page load.

Art:

Art drawn with Autodesk Sketchbook, © Starcloud.

Javascript:

Click the switch button. Image and background color must continuously switch 1, 2, 3, 1, 2, 3...

Note:

☀ If you have transform property in both the animation and the selector, they will conflict and the animation will be given the priority.

☀ "Cloud" is a class selector in this case. Therefore to have the javascript effective for the image swap, make sure to have the function execute for the number of images. Also note that to have javascript catch elements such as "body", need the [0] to function properly.

☀ Add some margin-bottom to give the scrollbar a break!
