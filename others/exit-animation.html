CSS hover exit animation tutorial

ホバー終了アニメーションについて

For exit animation, need to employ transitions on the main selector rather than the hover selector.

終了アニメーションを表示するには、「transition-duration」をホバーセレクターではなく、メインセレクターに着けることが必要です。

Transition on hover selector

ホバーセレクター

.c1 img:hover{
	transform: rotate(45deg);
	transition-duration: 1s;
}

Transition on main selector

メインセレクター

.c2 img:hover{
	transform: rotate(45deg);
}
.c2 img{
	transition-duration: 1s;
}
