Parallax Tutorial
Scroll down to see the magic...

Parallax scrolling is when a content moves as a different speed than another to create parallax effect in the webpage.

Basically, it is what you see when you ride on a train and see the trees in front of you zip past right in front of you while the mountains in the background moves as a much slower rate.

Ta-da!

In this simplified version, the background is fixed into position of the page, while the div moves with the scroll. (Essentially this is more of cool scrolling effect than actual parallax.) Set the image in a container, as a div, and set the height of the container. Fix the image in place so it does not move with the scroll.

No Javascript or other complex stuff, just CSS!

See the difference?

It could be done with just the height of the div with pixels, but if you want to cover the whole if the page (because responsive webpage is what I believe to be the industry standard nowadaysâ€¦), make sure to set the height of the div to 100%, and also you must apply height 100% to both the html and body in CSS.

The top and bottom image containers are set to 100% of the page height, while the middle two image containers are set to 400px each.
